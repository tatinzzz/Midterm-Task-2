Midterm-Task-2: Data Cleaning and Transformation Using Power Query Editor

  • This task demonstrates the process of data cleaning and preparation using Power Query. The dataset consists of multiple related tables, where cleaning techniques are applied to improve data quality and consistency before analysis.

    • Step-by-Step Process

    Step 1: Download and Load Data

    • Download the dataset (Uncleaned_DS_jobs.csv)
    • Open Excel
    • Go to Data → New Query → Open File → Text/CSV
    • Click Load and then Edit using Power Query Editor

    Step 2: Duplicate Raw Data

    • Right-click the dataset in the Queries pane
    • Select Duplicate

    Step 3: Clean Salary Data

    • Select the Salary Estimate column
    • Go to Transform Menu → Extract → Text Before Delimiter
    • Type "(" and click OK
    • Create two new columns: Min Salary and Max Salary
    • Repeat for Max Salary

    Step 4: Add Role Type Column

    • Go to Add Column Menu → Custom Column
    • Rename the column to "Role Type"
    • Use logic based on Job Title

    Step 5: Split Location Column

    • Select the Location column and add a Custom Column for corrections
    • Split the column by comma

    Step 6: Split Size Column

    • Create new columns for MinCompanySize and MaxCompanySize

    Step 7: Handle Negative Values

    • Filter out -1s from Competitors, 0s from Revenues, and -1s from Industry

    Step 8: Clean Company Names

    • Remove extra characters from company names

    Step 9: Copy Cleaning Steps as Proof

    • Copy the code from Advanced Editor and save it

    Step 10: Reshape and Group Data

    • Group by Role Type, Company Size, and State

    Step 11: Merge State Mapping

    • Merge State Full Name with dataset

    Step 12: Group by State

    • Group data by State and calculate average salary

    Step 13: View Query Dependencies

    • Check if all queries are correctly linked
